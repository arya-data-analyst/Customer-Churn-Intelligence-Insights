# Customer-Churn-Intelligence-Insights
This project focuses on analyzing customer churn behavior using an end-to-end data analytics pipeline. The goal is to identify patterns, risk factors, and key business insights that contribute to customer churn and help organizations make data-driven retention strategies.
----

----
📌 Project Overview

Customer churn is a major challenge for subscription-based businesses. Losing customers leads to revenue loss and reduced 

long-term profitability.


This project analyzes customer behavior, identifies churn drivers, and builds a predictive model to help businesses reduce

churn.

----
The project combines:

SQL for data cleaning and exploration

Python for analytics and modeling

Excel dashboard for visualization

The goal is to identify high-risk customers and recommend strategies to improve retention.

-----
🎯 Business Problem

The company is facing a ~27% churn rate, causing:


Revenue loss

Decreasing customer lifetime value

Customer migration to competitors

The objective of this project is to:


✔ Identify churn patterns

✔ Detect high-risk customer segments

✔ Analyze revenue impact

✔ Provide actionable business recommendations

🔍 Key Insights

Churn by Gender

Female customers → 64% of churn

Male customers → 36%

Churn by Contract Type

Contract Type	Churn Rate

Month-to-Month	46.5%

1-Year	11%

2-Year	2.7%

Long-term contracts significantly improve retention.


Churn by Internet Service

Service	Churn Rate

Fiber Optic	41.1%

Cable	25.7%

DSL	19.4%

No Internet	7.8%

Fiber plans may require service improvements.


Churn by Age

Customers above 50 years show the highest churn (~32%).


Younger customers show relatively lower churn.


Churn by Tenure

Customers with less than 12 months tenure show higher churn.


Customers staying more than 24 months are more stable.
